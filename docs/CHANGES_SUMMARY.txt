================================================================================
PATH ALIASES CONFIGURED
================================================================================

@           → ./src
@components → ./components  (supports both root and src)
@context    → ./context     (supports both root and src)
@data       → ./data        (supports both root and src)
@types      → ./types       (supports both root and src)

This allows builds to work with files in either location!

================================================================================
TROUBLESHOOTING
================================================================================

Error: Cannot find module './components/...'
Solution: npm run reorganize

Error: TypeScript compilation failed
Solution: npx tsc --noEmit (see errors)

Error: No dist directory
Solution: Check build output for errors

Error: Vercel build fails
Solution: Check Node version (use 18.x or 20.x)
